CLI command reference
# V1 — No time lock
sui client call \
  --package 0xc1222e16...6c2a7 \
  --module geopay_escrow \
  --function create_escrow \
  --args <COIN_ID> <SURVEYOR> <SURCON> "JOB-ID" "Desc" 500 0 \
  --gas-budget 10000000

# V2 — With agreed deadline (pass clock 0x6 + deadline timestamp in ms)
--function create_escrow_v2 \
  --args <COIN_ID> <SURVEYOR> <SURCON> "JOB-ID" "Desc" 500 0 0x6 <DEADLINE_MS>
# DEADLINE_MS = Unix timestamp in milliseconds of your chosen date

⚠️ Raise a dispute (client or surveyor)
⚠️
If there is a problem with the survey document or payment, either the client or surveyor can raise a dispute. This freezes the escrow — only SURCON/NIS can then resolve it with a custom percentage split via resolve_dispute.
# Client or Surveyor raises a dispute
sui client call \
  --package 0x537e7ab3...050d \
  --module geopay_escrow \
  --function raise_dispute \
  --args <ESCROW_ID> "reason for dispute"
